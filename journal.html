Week 1: Introduction to PWA (Mobile first design)

This week, I learned about the basics of Progressive Web Apps (PWA), including their features and benefits. I set up my development environment and created a simple PWA.

How did I approach mobile first design?
I started by designing the layout for small screens first using a single-colum layout. Then I used CSS media queries to make it responsiv for larger screens.

Most useful HTML or CSS concept:
I learned how to use flexbox for aligning navigation elements and layout control.

What was most challenging?
Understanding how to reuse the same navigationbar across multiple pages without JavaScript.

Week 2: Frontend Fundamentals

This week, I focused on enhancing the frontend of my PWA. I learned about semantic HTML5 elements and improved my CSS skills.

How did I approach mobile first design?
I built the layout to fit mobile screens first, then applied Grid and Flexbox for wider views.

Most useful concept:
Creating a consistent page structure using shared CSS and sematic HTML tags.

What was most challenging?
Keeping font size and padding consistent across devices.

Week 3: JavaScript & DOM Manipulation

this week, I delved into JavaScript basics and DOM manipulation to make my PWA interactive. I learned how to select and modify HTML elements using JavaScript

Which DOM selection method did you use?
getElementById() and querySelectorAll().

What was most Challenging?:
Making sure JS runs after HTML.

Debugging:
Used browser console.

Week 4: APIs (Storage, Browser, Third-Party)

This week, I explored various web APIs to enhance my PWA's functionality. I learned about Local Storage, Geolocation API, and Fetch API.

Which APIs did you use?
Local Storage (to save data), Clipboard API (to copy), and YouTube embed.

Challenging:
Converting data to JSON and back.
